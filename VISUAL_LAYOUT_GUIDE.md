## Button Anatomy

### Add Text Button (Primary)
```
┌──────────────────┐
│  + Add Text      │  ← Purple background (#6366f1)
└──────────────────┘     White text (#ffffff)
                         Rounded corners (6px)
                         Padding: 8px 14px
```

### Regular Button
```
┌──────────┐
│  [↶]     │  ← Dark background (#252932)
└──────────┘     Light text (#b4bac7)
                 Border: #2f3441
                 Hover: #2f3441 background
```

### Active Button (e.g., Bold is ON)
```
┌──────────┐
│  [B] ✓   │  ← Purple tint background
└──────────┘     Light purple text (#a5b4fc)
                 Purple border (#6366f1)
```

### Disabled Button
```
┌──────────┐
│  [🔒]    │  ← Grayed out (opacity: 0.35)
└──────────┘     No hover effect
                 Cursor: not-allowed
```
